# syfooprest

syfooprest er et REST-API i SBS som tilbyr oppfølgingsdialoger og oppfølgingsplaner til frontend-
applikasjonene i SBS (syfofront, syfoagfront).


## Lokal utvikling

Start opp via `LocalApplication.main`. Kjører på port 8580.
